# Xether AI - Data Infrastructure for AI

Xether AI automates the fragile parts of data work so AI teams can focus on outcomes. It provides infrastructure-grade reliability for collecting, cleaning, and versioning data.

![Xether AI Logo](/public/icon.svg)

## Features

- **Automated Data Cleaning**: Detect outliers and validate schemas automatically.
- **Dataset Version Control**: Treat data like code with branching and rollbacks.
- **Unified Ingestion**: Connectors for S3, Snowflake, Postgres, and more.
- **Synthetic Augmentation**: Fill data gaps with high-fidelity generation.
- **Enterprise Governance**: RBAC and audit logs for sensitive datasets.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Library**: React 19.2
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Components**: Shadcn/UI

## Getting Started

### Prerequisites

- Node.js 20.9 or later
- npm or pnpm

### Local Development

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

The project is optimized for deployment on [Vercel](https://vercel.com).
